1.Sistema de Gestión Hotelera
Aplicacion de escritorio para la gestión de reservas, check-in y check-out de un hotel.
Desarrollado siguiendo la artquitectura de 3 capas y seguridad y patrones de persistencia CSV.
2.Funcionalidades
-Gestión Visual Interfaz Intuitiva con Java Swing.
-Persitencia Guardado automatico de datos en Inventario_hotel.txt
-Reglas de Negocio Validación de disponiblidad y cálculo de costos.
3.Tecnologias
Java JDK 21
Java Swing (GUI)
Git / Gtihub
4.Como ejecutar 
Clonar el repositorio
Abrir vs code
Ejecutar java Main.java
